Explored how software maps to hardware — covering C programming, memory management, the x86-64 assembly language, processes, and debugging with GDB. Built a deep understanding of what happens beneath high-level code.

Programming Abstractions

Studied abstraction and algorithm design in C++ — including recursion, data structures (stacks, queues, trees, graphs), and sorting algorithms. Strengthened problem-solving skills through hands-on programming assignments.

C++ Data Structures Algorithms Recursion
EE 102A Electrical Engineering

Signal Processing & Linear Systems I

Analyzed continuous and discrete-time signals and systems using Fourier transforms, frequency response, and convolution. Applied concepts to filter design and system modeling using MATLAB and Julia.

MATLAB Julia Fourier Analysis Filter Design

Covered the fundamentals of circuit analysis including Kirchhoff's laws, nodal and mesh analysis, op-amps, and AC/DC circuit behavior. Developed hands-on experience with circuit simulation and lab measurements.
